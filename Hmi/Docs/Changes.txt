New Feature - Changes made to support splitting of the STATIC_FREQUENT_------_DEFAULT dataclass
              The STATIC_FREQUENT_------_DEFAULT dataclass contains HMI configuration data for all variants of HMI. For simple
              versions of HMI (i.e. without multiple operator pages or custom decimal places) much of the configuration is 
              redundant. The "SFD_SPLIT" pre-processor switch can be used to split the configuration data into three separate
              data classes. Hence redundant configuration data can be mapped to an area without NV coverage - to reduce NV requirement.
              NOTE: requires pre-processor directive SFD_SPLIT

NAME gbroarn
DETAILS
	Left/Right lockout operation modified
	To access the menus the user must press and hold the Up key until the Icons appear.
	The user must then release the Up key and press either Left or Right as appropriate within 3 sec.
	If the user continues to press the Up key, presses the Down key or fails to press either Left or Right
	then the Icons will dissapear and the procedure must be re-started. If at any time the exclamation mark
	icons appear the user must release all keys, wait 1 second and then restart the procedure.
